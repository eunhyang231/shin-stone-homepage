# 신주석재앙카 브랜드 가이드

## 프로젝트 목표
부모님 회사 신주석재앙카의 공식 홈페이지를 제작한다.
목표는 거래처가 봤을 때 신뢰감을 느끼는 기업 홈페이지를 만드는 것이다.

## 제작 기간
- 제작 기간: 7일
- 수정 및 테스트: 3일

## 브랜드 키워드
- 신뢰
- 견고함
- 정밀함
- 프리미엄
- 미니멀
- 건축자재 전문성

## 디자인 컨셉
- 화이트
- 차콜
- 네이비
- 스테인리스 실버
- 금속 질감
- 석재 텍스처
- 넓은 여백
- 현대적인 건축자재 회사 느낌

## 홈페이지 구성
- 홈
- 회사소개
- 제품소개
- 문의하기
- 관리자

## 제외할 것
- 쇼핑몰 결제 기능
- 회원가입
- 시공사례
- 과한 애니메이션
- 알록달록한 색상
- 영어 중심 디자인

## 디자인 원칙
1. 한국어를 중심으로 사용한다.
2. 부모님도 이해하고 관리할 수 있게 만든다.
3. 거래처가 전화 문의하기 쉽게 만든다.
4. 제품과 회사 신뢰도를 중심으로 구성한다.
5. 완성 후 네이버/구글 검색 등록까지 진행한다.
